## 🚀 快速开始

### 📦 安装运行
```bash
# 克隆项目
git clone [repository-url]

# 安装依赖
npm install

# 开发环境
npm run dev

# 构建桌面应用
npm run build
```

### 💻 系统要求
- **操作系统**：Windows 10+ / macOS 10.14+ / Ubuntu 18.04+
- **内存**：4GB RAM（推荐8GB）
- **存储**：2GB可用空间
