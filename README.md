## 🎯 **快速开始**

### 📥 **安装运行**（30秒完成）

```bash
# 1. 克隆项目
git clone https://github.com/your-username/AIStudioElectron.git
cd AIStudioElectron

# 2. 安装依赖
npm install

# 3. 启动开发环境
npm run dev
# 应用将自动打开 http://localhost:5173
```

### 🎮 **一键创作体验**（3分钟上手）

1. **打开应用**：`npm run dev` 启动后自动进入
2. **选择场景**：点击左侧菜单 **"内容创作" → "一键创作"**
3. **输入主题**：比如"中秋节月饼推荐"
4. **AI生成**：30秒内获得完整内容
5. **一键发布**：同步到抖音、小红书、B站
